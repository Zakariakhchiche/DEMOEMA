## 2. Ce qui tourne RÉELLEMENT

### 2.1 DEMOEMA app (`/root/DEMOEMA/`)

3 containers Docker, orchestration via `docker-compose.yml` + `docker-compose.override.yml` :

| Container | Image | Statut | Role |
|---|---|---|---|
| `demomea-caddy` | `caddy:2.8-alpine` | Up 53 min | Reverse proxy, ports 80/443 publics, TLS auto ACME |
| `demomea-backend` | build local FastAPI | Up 55 min (healthy) | API Python, expose 8000 interne |
| `demomea-frontend` | build local Next.js 15 | Up 3h | Frontend, expose 3000 interne |

Routing Caddy :
- `${DOMAIN}/api/*` → `backend:8000`
- `${DOMAIN}/*` → `frontend:3000`
- `studio.${DOMAIN}` → `supabase-studio:3000` avec `basic_auth admin`

### 2.2 Supabase self-hosted (`/root/supabase-proj/`)

Full stack **15 containers** Supabase auto-hébergés (variante Caddy) :

| Container | Role | Exposé ? |
|---|---|---|
| `supabase-db` | Postgres 15 + pgvector + pg_trgm (natif Supabase) | Non (via pooler) |
| `supabase-pooler` | Pooler Postgres | **5432 + 6543 publics** ⚠️ |
| `supabase-auth` | GoTrue JWT | Via kong |
| `supabase-studio` | UI admin | Via Caddy + basic_auth |
| `supabase-storage` | Object storage API | Via kong |
| `supabase-realtime` | WebSockets | Via kong |
| `supabase-kong` | API gateway | 8000 + 8443 publics ⚠️ |
| `supabase-meta`, `rest`, `analytics`, `vector`, `imgproxy`, `edge-functions` | services secondaires | Internes |

Réseau Docker `shared-supabase` (external, créé par supabase-proj) permet à `demomea-backend` + `demomea-caddy` de joindre `supabase-*` en interne.

## 6. Sécurité — points à traiter rapidement

1. ⚠️ **Ports 5432, 6543, 8000, 8443 exposés publiquement** (pooler Postgres + kong Supabase). À restreindre via UFW/iptables ou écouter sur 127.0.0.1 uniquement.
2. ⚠️ **Pas d'UFW activé** (à vérifier). Installer et configurer `ufw default deny incoming; ufw allow 22,80,443`.
3. ⚠️ **fail2ban** non vérifié. À installer.
4. ⚠️ **Backups automatiques absents**. Risque majeur → SCRUM-91 priorité.
5. ⚠️ **unattended-upgrades** non vérifié. À configurer pour patches sécurité auto.
6. ⚠️ **Hash bcrypt studio** en clair dans Caddyfile committé (OK car coût bcrypt 14, mais mieux via env var).
